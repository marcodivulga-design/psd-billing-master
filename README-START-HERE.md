## 🔧 Configuração Rápida

### 1. Banco de Dados

```bash
# Criar banco de dados
mysql < drizzle/migrations/0001-initial-schema.sql

# Ou usar Docker (já incluído no docker-compose.yml)
docker-compose up -d mysql
```

### 2. Variáveis de Ambiente

```bash
# Copiar exemplo
cp .env.example .env

# Editar com suas credenciais
nano .env
```

**Variáveis Necessárias:**
```
DATABASE_URL=mysql://user:password@localhost/psd_hub
STRIPE_API_KEY=sk_test_...
STRIPE_WEBHOOK_SECRET=whsec_...
ASAAS_API_KEY=...
SENDGRID_API_KEY=...
OPENAI_API_KEY=...
```

### 3. Instalar Dependências

```bash
npm install
# ou
pnpm install
```

### 4. Iniciar Desenvolvimento

```bash
npm run dev
# ou
pnpm dev
```

### 5. Acessar

```
Frontend: http://localhost:3000
API: http://localhost:3000/api
Admin: http://localhost:3000/admin
```

## 🆘 Troubleshooting

### Erro: "Cannot find module"

```bash
# Limpar node_modules
rm -rf node_modules
npm install
```

### Erro: "Database connection failed"

```bash
# Verificar MySQL
mysql -u root -p
# Criar banco
CREATE DATABASE psd_hub;
# Executar migrations
mysql psd_hub < drizzle/migrations/0001-initial-schema.sql
```

### Erro: "Port 3000 already in use"

```bash
# Matar processo
lsof -ti:3000 | xargs kill -9
# Ou usar porta diferente
PORT=3001 npm run dev
```

### Erro: "Stripe API key invalid"

```bash
# Verificar .env
cat .env | grep STRIPE
# Atualizar com chave correta
nano .env
```
