# 👨‍💼 Guia Completo do Administrador

## 🎯 Bem-vindo ao Painel Admin

Este guia vai ajudá-lo a gerenciar sua loja CACHA-A-CUTELARIA.

---

## 📊 Dashboard Principal

### Acessar Admin

```
1. Faça login com sua conta admin
2. Clique em "Admin" no menu
3. Você verá o dashboard principal
```

### Widgets Principais

```
Dashboard mostra:
├── Vendas (últimos 7 dias)
├── Receita (últimos 7 dias)
├── Pedidos (últimos 7 dias)
├── Clientes (últimos 7 dias)
├── Taxa de conversão
├── Ticket médio
├── Clientes ativos
└── Produtos mais vendidos
```

---

## 📦 Gerenciamento de Produtos

### Adicionar Novo Produto

```
1. Clique em "Produtos"
2. Clique em "Adicionar Novo"
3. Preencha:
   ├── Nome
   ├── Descrição
   ├── Preço
   ├── Custo
   ├── SKU
   ├── Categoria
   ├── Tamanho
   ├── Tipo de cabo
   ├── Tipo de aço
   ├── Peso
   ├── Dimensões
   ├── Stock
   └── Fotos (mínimo 3)
4. Clique em "Publicar"
```

### Editar Produto

```
1. Clique em "Produtos"
2. Clique no produto
3. Faça as alterações
4. Clique em "Salvar"
```

### Deletar Produto

```
1. Clique em "Produtos"
2. Clique no produto
3. Clique em "Deletar"
4. Confirme
```

### Gerenciar Estoque

```
1. Clique em "Produtos"
2. Clique em "Estoque"
3. Você verá:
   ├── Produtos com baixo estoque
   ├── Produtos fora de estoque
   ├── Histórico de movimentações
   └── Previsão de estoque
4. Clique em "Adicionar Estoque" para repor
```

### Gerenciar Fotos

```
1. Clique em "Produtos"
2. Clique no produto
3. Clique em "Fotos"
4. Você pode:
   ├── Adicionar nova foto
   ├── Reordenar fotos
   ├── Deletar foto
   ├── Definir como principal
   └── Editar descrição
```

## 📋 Gerenciamento de Pedidos

### Ver Pedidos

```
1. Clique em "Pedidos"
2. Você verá:
   ├── Número do pedido
   ├── Cliente
   ├── Data
   ├── Status
   ├── Total
   └── Ações
```

### Filtrar Pedidos

```
Filtros disponíveis:
├── Status (Pendente, Processando, Enviado, Entregue)
├── Data (Hoje, Últimos 7 dias, Últimos 30 dias)
├── Cliente
├── Faixa de preço
└── Método de pagamento
```

### Ver Detalhes do Pedido

```
1. Clique no pedido
2. Você verá:
   ├── Informações do cliente
   ├── Itens do pedido
   ├── Endereço de entrega
   ├── Método de pagamento
   ├── Status do pedido
   ├── Histórico de movimentações
   ├── Notas internas
   └── Ações disponíveis
```

### Processar Pagamento

```
1. Clique no pedido
2. Clique em "Processar Pagamento"
3. Verifique:
   ├── Método de pagamento
   ├── Valor
   ├── Status
4. Clique em "Confirmar"
```

### Enviar Pedido

```
1. Clique no pedido
2. Clique em "Enviar"
3. Selecione:
   ├── Transportadora
   ├── Tipo de frete
   ├── Código de rastreamento
4. Clique em "Confirmar"
5. Cliente receberá notificação
```

### Cancelar Pedido

```
1. Clique no pedido
2. Clique em "Cancelar"
3. Selecione motivo
4. Clique em "Confirmar"
5. Reembolso será processado
```

## 💳 Gerenciamento de Pagamentos

### Ver Pagamentos

```
1. Clique em "Pagamentos"
2. Você verá:
   ├── Transações recentes
   ├── Status de cada transação
   ├── Valor
   ├── Método
   └── Data
```

### Processar Reembolso

```
1. Clique em "Pagamentos"
2. Clique na transação
3. Clique em "Reembolsar"
4. Selecione:
   ├── Valor (total ou parcial)
   ├── Motivo
5. Clique em "Confirmar"
6. Cliente receberá reembolso em 3-5 dias úteis
```

### Reconciliar Contas

```
1. Clique em "Pagamentos"
2. Clique em "Reconciliação"
3. Você verá:
   ├── Saldo esperado
   ├── Saldo real
   ├── Diferenças
4. Clique em "Reconciliar"
```

## 👥 Gerenciamento de Clientes

### Ver Clientes

```
1. Clique em "Clientes"
2. Você verá:
   ├── Nome
   ├── Email
   ├── Telefone
```

### Filtrar Clientes

```
Filtros disponíveis:

├── Faixa de gasto
├── Data de cadastro

```

### Ver Detalhes do Cliente

```
1. Clique no cliente
2. Você verá:
   ├── Informações pessoais
   ├── Histórico de pedidos
   ├── Histórico de pagamentos
   ├── Endereços salvos
   ├── Wishlist
   ├── Pontos de lealdade
   ├── Badges
   ├── Referrals
   └── Notas internas
```

### Enviar Mensagem

```
1. Clique no cliente
2. Clique em "Enviar Mensagem"
3. Selecione canal:
   ├── Email
   ├── SMS
   ├── Push
   └── WhatsApp
4. Escreva a mensagem
5. Clique em "Enviar"
```

### Bloquear Cliente

```
1. Clique no cliente
2. Clique em "Bloquear"
3. Selecione motivo
4. Clique em "Confirmar"
5. Cliente não poderá fazer compras
```

## 🏪 Gerenciamento de Marketplace

### Onboarding de Vendors

```
1. Clique em "Marketplace"
2. Clique em "Vendors"
3. Clique em "Novo Vendor"
4. Preencha:
   ├── Nome da loja
   ├── Email
   ├── Telefone
   ├── Documentos
   ├── Conta bancária
   ├── Comissão
5. Clique em "Enviar Convite"
6. Vendor receberá email com link
```

### Aprovar/Rejeitar Vendor

```
1. Clique em "Marketplace"
2. Clique em "Vendors Pendentes"
3. Clique no vendor
4. Revise:
   ├── Documentos
   ├── Conta bancária
   ├── Políticas
5. Clique em "Aprovar" ou "Rejeitar"
```

### Gerenciar Comissões

```
1. Clique em "Marketplace"
2. Clique em "Comissões"
3. Você pode:
   ├── Ver comissões por vendor
   ├── Ajustar taxa de comissão
   ├── Ver histórico
   └── Processar payouts
```

### Processar Payouts

```
1. Clique em "Marketplace"
2. Clique em "Payouts"
3. Você verá:
   ├── Vendors com saldo
   ├── Valor a pagar
   ├── Método de pagamento
4. Clique em "Processar Payout"
5. Confirme
6. Vendor receberá pagamento em 3-5 dias úteis
```
