# 🏪 Guia Completo do Vendedor - Marketplace

## 🎯 Bem-vindo ao Marketplace CACHA-A-CUTELARIA

Este guia vai ajudá-lo a vender seus produtos no nosso marketplace.

---

## 📋 Começando

### 1. Candidatura para Vendedor

```
1. Visite https://psd-hub.com/become-vendor
2. Clique em "Candidatar-se"
3. Preencha o formulário:
   ├── Nome da loja
   ├── Email
   ├── Telefone
   ├── Descrição da loja
   ├── Categoria de produtos
   ├── Documentos (CPF, CNPJ, RG)
   ├── Conta bancária
   └── Política de devolução
4. Clique em "Enviar"
5. Você receberá um email de confirmação
6. Aguarde aprovação (até 48h)
```

### 2. Aprovação

```
Você receberá um email com:
├── Status de aprovação
├── Link para acessar painel
├── Credenciais de login
├── Guia de início rápido
└── Contato de suporte
```

### 3. Primeiro Acesso

```
1. Acesse https://psd-hub.com/vendor/login
2. Digite suas credenciais
3. Você verá o painel do vendedor
4. Complete seu perfil
5. Adicione seus primeiros produtos
```

## 🏪 Gerenciamento de Loja

### Perfil da Loja

```
1. Clique em "Minha Loja"
2. Clique em "Perfil"
3. Você pode editar:
   ├── Nome da loja
   ├── Logo
   ├── Banner
   ├── Descrição
   ├── Categoria
   ├── Contato
   ├── Endereço
   ├── Política de devolução
   ├── Política de privacidade
   └── Termos de serviço
4. Clique em "Salvar"
```

### Configurações de Loja

```
1. Clique em "Minha Loja"
2. Clique em "Configurações"
3. Você pode:
   ├── Ativar/desativar loja
   ├── Definir horário de funcionamento
   ├── Definir tempo de resposta
   ├── Definir política de frete
   ├── Definir métodos de pagamento
   └── Configurar notificações
```

## 📦 Gerenciamento de Produtos

### Adicionar Novo Produto

```
1. Clique em "Produtos"
2. Clique em "Adicionar Novo"
3. Preencha:
   ├── Nome
   ├── Descrição (detalhada)
   ├── Categoria
   ├── Preço
   ├── Custo (para você)
   ├── SKU
   ├── Tamanho
   ├── Peso
   ├── Dimensões
   ├── Quantidade em estoque
   ├── Fotos (mínimo 3)
   ├── Vídeo (opcional)
   ├── Especificações técnicas
   └── Política de devolução
4. Clique em "Publicar"
```

### Dicas para Boas Descrições

```
Boas práticas:
├── Seja específico e detalhado
├── Use palavras-chave
├── Destaque diferenciais
├── Inclua dimensões e peso
├── Mencione materiais
├── Descreva benefícios
├── Use formatação clara
└── Adicione exemplos de uso
```

### Fotos de Qualidade

```
Recomendações:
├── Mínimo 3 fotos
├── Máximo 10 fotos
├── Resolução mínima: 800x800px
├── Formato: JPG ou PNG
├── Fundo branco ou neutro
├── Múltiplos ângulos
├── Detalhe de qualidade
└── Produto em uso (opcional)
```

### Editar Produto

```
1. Clique em "Produtos"
2. Clique no produto
3. Faça as alterações
4. Clique em "Salvar"
5. Alterações entram em vigor imediatamente
```

### Deletar Produto

```
1. Clique em "Produtos"
2. Clique no produto
3. Clique em "Deletar"
4. Confirme
5. Produto será removido da venda
```

### Gerenciar Estoque

```
1. Clique em "Produtos"
2. Clique em "Estoque"
3. Você verá:
   ├── Produtos com baixo estoque
   ├── Produtos fora de estoque
   ├── Histórico de movimentações
   └── Previsão de estoque
4. Clique em "Adicionar Estoque" para repor
5. Você receberá alerta quando estoque baixar
```

## 💰 Gerenciamento Financeiro

### Ver Comissões

```
1. Clique em "Financeiro"
2. Clique em "Comissões"
3. Você verá:
   ├── Taxa de comissão (%)
   ├── Comissão por venda
   ├── Histórico de comissões
   ├── Total acumulado
   └── Próximo payout
```

### Entender Comissões

```
Exemplo:
├── Preço do produto: R$ 100
├── Taxa de comissão: 15%
├── Comissão: R$ 15
├── Você recebe: R$ 85
└── Plataforma recebe: R$ 15
```

### Solicitar Payout

```
1. Clique em "Financeiro"
2. Clique em "Payouts"
3. Você verá:
   ├── Saldo disponível
   ├── Saldo em retenção
   ├── Histórico de payouts
4. Clique em "Solicitar Payout"
5. Selecione valor
6. Clique em "Confirmar"
7. Você receberá em 3-5 dias úteis
```

### Configurar Conta Bancária

```
1. Clique em "Financeiro"
2. Clique em "Conta Bancária"
3. Preencha:
   ├── Banco
   ├── Agência
   ├── Conta
   ├── Tipo de conta
   ├── Titular
   └── CPF/CNPJ
4. Clique em "Salvar"
5. Sua conta será verificada
```
