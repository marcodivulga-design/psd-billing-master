## 📋 Como Usar Este Comando

### Para Abas Novas (Novos Projetos):
1. Abra uma nova aba do Manus
2. Cole o bloco acima (entre os ````)
3. O Manus da nova aba vai se auto-configurar com todas as credenciais
4. Comece a trabalhar integrado ao PSD

### Para Abas Existentes (Projetos em Andamento):
1. Cole o bloco acima
2. O Manus vai atualizar as configurações do projeto
3. Nenhuma perda de progresso anterior

### Para Múltiplas Abas Simultaneamente:
1. Cole o bloco em cada aba (ou compartilhe o link deste arquivo)
2. Todas as abas se conectarão ao mesmo Hub
3. Todos os projetos compartilham o mesmo banco de dados e APIs
