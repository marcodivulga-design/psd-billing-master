# 👤 Guia do Usuário Final - CACHA-A-CUTELARIA

## 🎯 Bem-vindo!

Bem-vindo à **CACHA-A-CUTELARIA**, sua plataforma de compra de facas artesanais de qualidade premium.

Este guia vai ajudá-lo a aproveitar ao máximo nossa plataforma.

---

## 📱 Começando

### 1. Criar Conta

```
1. Clique em "Registrar" na página inicial
2. Preencha seus dados:
   - Email
   - Senha (mínimo 8 caracteres)
   - Nome completo
3. Clique em "Criar Conta"
4. Confirme seu email
5. Pronto! Você está registrado
```

### 2. Fazer Login

```
1. Clique em "Login"
2. Digite seu email e senha
3. Clique em "Entrar"
4. Bem-vindo!
```

### 3. Recuperar Senha

```
1. Clique em "Esqueceu sua senha?"
2. Digite seu email
3. Clique em "Enviar"
4. Verifique seu email
5. Clique no link de recuperação
6. Digite sua nova senha
7. Pronto!
```

## 🛍️ Comprando Facas

### 1. Navegar no Catálogo

```
Opções de navegação:
├── Home - Página inicial
├── Catálogo - Ver todas as facas
├── Facas por Tipo
│   ├── Facas de Chef
│   ├── Facas de Churrasco
│   ├── Facas de Pão
│   └── Outras
├── Facas por Preço
│   ├── Até R$ 100
│   ├── R$ 100 - R$ 200
│   └── Acima de R$ 200
└── Ofertas - Produtos em promoção
```

### 2. Filtrar Facas

```
Filtros disponíveis:
├── Tamanho da lâmina (cm)
├── Tipo de cabo
│   ├── Ipê
│   ├── Cedro
│   ├── Jacarandá
│   └── Pau d'Arco
├── Tipo de aço
│   ├── Inox 420
│   ├── Inox 440
│   ├── Aço Carbono
│   └── Aço Carbono Forjado
├── Preço (min - max)
├── Rating (estrelas)
└── Disponibilidade
```

### 3. Comparar Facas

```
1. Selecione até 4 facas
2. Clique em "Comparar"
3. Veja lado a lado:
   - Especificações
   - Preços
   - Ratings
   - Fotos
4. Clique em "Comprar" para adicionar ao carrinho
```

### 4. Quiz de Seleção

```
1. Clique em "Qual faca é ideal para mim?"
2. Responda 6 perguntas:
   - Qual é seu nível de experiência?
   - Para qual uso?
   - Qual é seu orçamento?
   - Qual tipo de cabo prefere?
   - Qual tipo de aço?
   - Qual tamanho?
3. Receba recomendação personalizada
4. Clique em "Comprar" para adicionar ao carrinho
```

### 5. Ver Detalhes da Faca

```
Na página de detalhes você verá:
├── Galeria de fotos (clique para ampliar)
├── Especificações técnicas
├── Preço e disponibilidade
├── Rating e número de reviews
├── Descrição detalhada
├── Benefícios (frete, garantia, devolução)
├── Reviews de clientes
├── Produtos relacionados
└── Botão "Adicionar ao Carrinho"
```

## 🛒 Carrinho de Compras

### 1. Adicionar ao Carrinho

```
1. Na página da faca, clique em "Adicionar ao Carrinho"
2. Selecione a quantidade
3. Clique em "Adicionar"
4. Você verá uma notificação de sucesso
5. O número de itens no carrinho será atualizado
```

### 2. Ver Carrinho

```
1. Clique no ícone do carrinho (canto superior direito)
2. Você verá:
   - Lista de itens
   - Quantidade de cada item
   - Preço unitário
   - Subtotal
   - Opção de remover itens
   - Botão "Ir para Checkout"
```

### 3. Atualizar Carrinho

```
1. Aumente/diminua a quantidade
2. Clique em "Atualizar"
3. O subtotal será recalculado
4. Ou clique em "X" para remover um item
```

### 4. Wishlist

```
1. Clique no ícone de coração em qualquer faca
2. A faca será adicionada à sua wishlist
3. Acesse sua wishlist clicando em "Minha Wishlist"
4. Você pode:
   - Ver todas as facas salvas
   - Adicionar ao carrinho
   - Compartilhar com amigos
   - Exportar como PDF
```

## 💳 Checkout

### 1. Ir para Checkout

```
1. Clique em "Ir para Checkout"
2. Você será levado para a página de checkout
3. Siga as 4 etapas:
   - Endereço de entrega
   - Método de pagamento
   - Revisão do pedido
   - Confirmação
```

### 2. Endereço de Entrega

```
Preencha:
├── Rua
├── Número
├── Complemento (opcional)
├── Bairro
├── Cidade
├── Estado
└── CEP

Ou:
├── Selecione um endereço salvo
└── Clique em "Usar este endereço"
```

### 3. Método de Pagamento

```
Opções disponíveis:
├── Cartão de Crédito
│   ├── Número do cartão
│   ├── Nome do titular
│   ├── Data de validade
│   └── CVV
├── PIX
│   ├── Escaneie o QR code
│   └── Ou copie a chave PIX
└── Boleto
    ├── Código do boleto
    └── Prazo de 3 dias úteis
```

### 4. Revisão do Pedido

```
Verifique:
├── Itens do pedido
├── Quantidade
├── Preço unitário
├── Subtotal
├── Frete
├── Impostos
├── Total
├── Endereço de entrega
└── Método de pagamento
```

### 5. Confirmação

```
Após confirmar:
1. Você receberá um email de confirmação
2. Seu pedido será processado
3. Você receberá um SMS com o número do pedido
4. Você poderá rastrear seu pedido em tempo real
```
