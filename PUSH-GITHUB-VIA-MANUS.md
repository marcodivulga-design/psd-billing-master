## 📋 OPÇÃO 1: PUSH VIA MANUS MANAGEMENT UI (Recomendado)

### Passo 1: Abra o Manus Management UI

1. Clique no botão **"Management UI"** (canto superior direito)
2. Ou clique no card do projeto

### Passo 2: Vá para Settings → GitHub

1. No Management UI, clique **"Settings"** (engrenagem)
2. Procure por **"GitHub"** ou **"Export"**
3. Clique **"Export to GitHub"**

### Passo 3: Configure o Export

1. **Owner:** `marcodivulga-design`
2. **Repository:** `psd-billing-master`
3. **Branch:** `master`
4. Clique **"Export"** ou **"Publish"**

### Resultado

✅ Código será enviado para GitHub automaticamente  
✅ Vercel fará deploy automático  
✅ URL pública será gerada
